What is token name? The token\'s name is any text or file. The name is a source of the token\'s hash.

Can the name be arbitrary data? No, it has to be simple ASCII textual data, not too long.

What if the source of the hash is large or binary? Use the file loading option (File button); or compute the hash externally and use it instead of the name.

Is the name open to the public? Yes, if it is a simple short text.

How to hide the name (make it not public)? Use the hash directly instead of the name.
